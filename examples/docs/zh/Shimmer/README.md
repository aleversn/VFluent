### Propoties
---
| 属性(attr) |             类型(type)             | 必填(required) | 默认值(default) | 说明(statement) |
|:----------:|:----------------------------------:|:--------------:|:---------------:|:---------------:|
|     theme     | String |       No       |     system      |       主题样式, 包含`light`, `dark`, `system`, `custom`几种样式              |

### Slot
---
1. Default

默认样式为:

```vue
<div class="sample"></div>
```

自定义`Shimmer`内容时, 将要呈现Skeleton效果的元素加上`sample`类.

```javascript
<fv-shimmer>
    <div class="sample"></div>
</fv-shimmer>
```
